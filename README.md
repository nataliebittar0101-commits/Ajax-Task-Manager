# Ajax Task Manager

A task management web application using Ajax to add tasks and update task status dynamically without page refresh.

## Description

This project is a simple task management system that allows users to add new tasks and update the status of existing tasks dynamically.

The main goal of the project is to demonstrate how Ajax can be used to communicate with the server without refreshing the page.  
Tasks are stored in a MySQL database and updated through PHP backend files.

## Features

- Add new tasks using Ajax
- Update task status dynamically
- Store tasks in a MySQL database
- Display tasks from the database
- Handle client-server communication without page refresh
- Simple and clean task management interface
